Desain UI Pengingat Jadwal

Project ini adalah perancangan User Interface (UI) untuk aplikasi Pengingat Jadwal Minum Obat yang bertujuan membantu pengguna agar tidak melewatkan jadwal penting mereka.

Desain berfokus pada tampilan yang bersih, modern, dan mudah digunakan. Aplikasi ini dirancang untuk membantu pasien atau pengguna agar selalu tepat waktu dalam mengonsumsi obat sesuai jadwal yang telah diatur.

Fitur yang dirancang dalam UI:

  • Halaman onboarding pengguna
  • Halaman input jadwal pengingat
  • Notifikasi pengingat
  • Dashboard list jadwal
  • Profil pengguna

Peran: UI/UX Designer
