Desain UI Sistem Pencatatan Hutang

Project ini adalah desain User Interface (UI) untuk aplikasi Sistem Pencatatan Hutang berbasis web. Tujuannya untuk memudahkan pengguna dalam mencatat, memantau, dan mengelola hutang-piutang secara rapi.

Desain UI dibuat dengan tampilan yang modern, simple, dan mudah digunakan bahkan oleh pengguna awam.

Fitur yang dirancang dalam UI:

  • Dashboard total hutang dan piutang
  • Form input data hutang/piutang
  • Riwayat transaksi
  • Notifikasi pengingat pembayaran
  • Profil pengguna

Tools: Figma
Peran: UI/UX Designer
